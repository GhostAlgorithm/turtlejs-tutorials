Tutorial 4: Moviendo la tortuga.

En este tutorial se muestra cómo puedes mover a la tortuga usando bloques de la paleta Básica o paleta Tortuga.

Para este tutorial es necesario que hayas leído el tutorial 2 y el tutorial 3, de esta forma tengas menos dudas de lo que en este se realizará.


Usando el bloque Forward

Por medio de este bloque, se puede mover a la tortuga hacia adelante, pero dependiendo de hacia donde la cabeza señale.

Requerido para su funcionamiento:

Para su funcionamiento, se necesita de un bloque de valor, debido a que sin este no se puede especificar cuanto será el movimiento. El movimiento en este caso es en base a pixeles en el lienzo. Tomando como referencia que la tortuga estará siempre en una posición 0.


Usando el bloque Backward

Este bloque permite mover la tortuga hacia atrás de donde esta se encuentre y dependiendo de dónde su cabeza apunte.

Requerido para su funcionamiento:

Para que este bloque funcione, se necesita de un bloque de valor, en este caso para especificar cuantos pixeles la tortuga deberá moverse.


Usando Left

Con este bloque se puede rotar a la tortuga hacia la izquierda tantos grados se quieran.

Requerido para su funcionamiento:

Este bloque también requiere de un bloque de valor, en su caso, para especificar cuantos grados positivos se desea rotar la tortuga.


Usando Right

Con este bloque se puede rotar a la tortuga hacia la derecha cuantos grados se deseen.

Requerido para su funcionamiento:

Al igual que el bloque Left, necesita de un bloque de valor para funcionar, siendo este bloque la variable que especifique la cantidad de grados positivos a rotar hacia la derecha.


Usado Arc

Con este bloque se puede mover a la tortuga en puras líneas curvas hasta figuras circulares y elípticas.

Requerido para su funcionamiento:

Para usar este bloque se necesitan dos bloques de valor, uno para indicar el ángulo de inclinación y otro bloque de valor para indicar el tamaño del radio a utilizar.


Usando Set XY

Por medio de este bloque se puede especificar un movimiento directo de la tortuga hasta una posición definida, tomando en cuenta la posición en el eje -x y en el eje –y.

Requerido para su funcionamiento:

Para poder usar este bloque se necesitan dos diferentes bloques de valor, uno para especificar el eje –x y otro para especificar el eje –y. el movimiento se hará desde un punto 0,0 desde donde en ese momento se encuentre la tortuga.


Usando Clean

Permite borrar cualquier dibujo que se encuentre dentro del lienzo.

Requerido para su funcionamiento:

Ninguno, al ejecutarse todos los bloques y llegar al bloque Clean, esté borrará todo y lo dejará en su forma por default.
