Tutorial 3 Paleta Básica (Paleta Tortuga)

En este tutorial se muestran los bloques dentro de la paleta Tortuga y su función.




Bloque Forward: Con este bloque se puede mover a la tortuga hacia delante de donde se su cabeza indique, al moverse la tortuga, se irá dibujando. Para usar este bloque es necesaria la unión con el bloque de valor de esta forma la aplicación sabrá que tanto la tortuga se moverá.


Bloque Backward: Al igual que con el bloque forward, dependiendo de hacía a donde apunte la cabeza de la tortuga, así será su movimiento, en este caso moviéndose hacia atrás. También se necesita del bloque de valor para especificar cuanto se desea mover.


Bloque Right: Este bloque permite rotar la tortuga hacia la derecha tantos grados positivos desees.


Bloque Left: Este bloque permite rotar la tortuga al lado izquierdo, tantos grados positivos desees.


Bloque de valor (bloque 100): Es el bloque que permite guardar un valor para ser usado como una variable.


Bloque Clean: Es un bloque que al igual que el borrador, permite borrar completamente el dibujo trazado por la tortuga y además la deja en el lugar predeterminado, que es el centro de la pantalla.


Bloque Heading: Por medio de este bloque se puede conocer la rotación absoluta de la inclinación de la cabeza de la tortuga.


Bloque Set heading: Se utiliza este bloque para especificar la rotación absoluta de la tortuga.


Bloque Arc: Este bloque permite lineas curvas, para esto requiere de la ayuda de dos bloques de valor, uno para indicar el ángulo y otro para indicar el radio por el cual se moverá.


Bloque xcor: Este bloque permite obtener la posición en el eje x de donde se encuentra la tortuga.


Bloque ycor: Este bloque permite obtener la posición en el eje y de donde se encuentra la tortuga.


Bloque set xy: Por medio de este bloque, se puede ubicar a la tortuga en un lugar específico y a la vez trazar una línea desde la posición de origen hasta la posición inicial. Para usar este bloque necesitas dos diferentes valores, uno para la posición en x y el otro para la posición en y, ambos en sus respectivos ejes.
